# SKMS Talk Agent — 포트폴리오 요약

## 프로젝트 개요
SK그룹 SUPEX추구협의회를 위한 AI 경영철학 에이전트. 선대회장님 경영철학(SKMS, 선경실록)을 RAG 기반으로 검색·답변하고, AI 발표자료를 자동 생성하며, 선대회장님 음성을 AI로 복원하는 시스템.

## 핵심 기능
- **AI 채팅 에이전트**: 7만5천건 SKMS 문서 RAG 검색 + 선대회장님 화법 답변
- **발표자료 자동 생성**: 10장 슬라이드 + 사회자 멘트 + 선대회장님 육성 TTS
- **AI 음성 클로닝**: ElevenLabs PVC로 선대회장님 실제 녹취 학습·복원
- **AI 심사위원**: 3인 패널 토론 + 심사위원장 종합 평가
- **슬라이드 편집**: 목차 모달에서 순서변경/추가/삭제, 일반/육성 슬라이드 선택

## 기술 스택
- Next.js 16 (App Router) / TypeScript / Tailwind CSS v4
- Claude Sonnet 4 (Anthropic) / Voyage AI (임베딩) / ElevenLabs (TTS)
- Supabase PostgreSQL + pgvector (74,916 벡터 청크)
- Netlify (배포) → SK VDI 내부망 이관 예정
